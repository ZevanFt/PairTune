## 一、项目概述

### 1.1 产品定位

| 项目 | 内容 |
|------|------|
| **中文名** | 合拍 |
| **英文名** | PairTune |
| **品牌语** | 两个人，更容易坚持 |
| **类型** | 双人协作成长工具 |
| **核心价值** | 把口头约定变成可执行、可激励、可复盘的双人协作闭环 |

### 1.2 目标用户

- 对象（情侣）
- 闺蜜
- 室友
- 学习搭子
- 同事搭档

### 1.3 核心功能

| 功能模块 | 描述 |
|----------|------|
| **任务** | 四象限任务、截止日期、积分奖励、完成状态 |
| **商城** | 发布奖励、兑换奖励、库存管理、已兑记录 |
| **通知** | 任务提醒、兑换提醒、关系动态、系统消息 |
| **我的** | 资料编辑、关系管理、账号安全、通知设置 |

## 二、设计理念

### 2.1 视觉关键词

```
温暖 · 协作 · 成长 · 轻仪式感
```

### 2.2 设计原则

1. **极简主义** - 大量留白，干净的界面
2. **微妙边框** - 使用细边框而非阴影创造层次
3. **克制色彩** - 主色用于强调，大量使用灰色系
4. **清晰层次** - 通过背景色差异区分信息层级
5. **精致交互** - 微妙的hover效果，平滑的过渡

### 2.3 设计参考

- **Linear** - 极简、专业、现代
- **Notion** - 清晰、文档化
- **Things 3** - 优雅、任务导向

## 三、色彩系统

### 3.1 主色

| 用途 | 浅色模式 | 暗色模式 | 说明 |
|------|----------|----------|------|
| **主色** | `#5E6AD2` | `#7B83EB` | Linear紫，用于强调、链接、按钮 |
| **主色浅** | `#EEF0FD` | `#2A2F4A` | 主色背景 |

### 3.2 文字色

| 用途 | 浅色模式 | 暗色模式 | 说明 |
|------|----------|----------|------|
| **主文字** | `#1A1A1A` | `#F5F5F5` | 标题、重要内容 |
| **次要文字** | `#5C5C5C` | `#B3B3B3` | 正文、描述 |
| **三级文字** | `#8A8A8A` | `#8A8A8A` | 辅助信息 |
| **弱化文字** | `#B3B3B3` | `#666666` | 占位符、禁用 |

### 3.3 背景色

| 用途 | 浅色模式 | 暗色模式 | 说明 |
|------|----------|----------|------|
| **画布** | `#FFFFFF` | `#0A0A0A` | 页面背景 |
| **表面** | `#FFFFFF` | `#141414` | 卡片背景 |
| **悬停** | `#FAFAFA` | `#1A1A1A` | 悬停状态 |
| **激活** | `#F5F5F5` | `#242424` | 激活状态 |
| **弱化** | `#F0F0F0` | `#2A2A2A` | 次级背景 |

### 3.4 边框色

| 用途 | 浅色模式 | 暗色模式 | 说明 |
|------|----------|----------|------|
| **默认** | `#E5E5E5` | `#2A2A2A` | 卡片边框 |
| **悬停** | `#D4D4D4` | `#3A3A3A` | 悬停边框 |
| **聚焦** | `#5E6AD2` | `#7B83EB` | 输入框聚焦 |

### 3.5 语义色

| 用途 | 颜色 | 浅色背景 | 说明 |
|------|------|----------|------|
| **成功** | `#4CAF50` | `#E8F5E9` | 完成、正确 |
| **警告** | `#FF9800` | `#FFF3E0` | 提醒、注意 |
| **危险** | `#E53935` | `#FFEBEE` | 错误、删除 |
| **信息** | `#2196F3` | `#E3F2FD` | 提示、链接 |

### 3.6 四象限色

| 象限 | 颜色 | 说明 |
|------|------|------|
| **紧急重要** | `#E53935` | 红色，立即执行 |
| **重要不紧急** | `#5E6AD2` | 主色，规划执行 |
| **紧急不重要** | `#FF9800` | 橙色，委托处理 |
| **不紧急不重要** | `#8A8A8A` | 灰色，稍后处理 |

## 九、动效规范

### 9.1 时长

| 名称 | 值 | 用途 |
|------|-----|------|
| **fast** | 100ms | 微交互 |
| **normal** | 200ms | 常规过渡 |
| **slow** | 300ms | 大动作 |

### 9.2 缓动曲线

- **标准**: `ease-out`
- **弹入**: `ease-out-back`
- **弹出**: `ease-in`

### 9.3 动效场景

| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮点击 | 100ms | ease-out |
| 卡片展开 | 200ms | ease-out |
| 页面切换 | 300ms | ease-out |
| 底部面板弹出 | 300ms | ease-out |
