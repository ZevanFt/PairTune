# 合拍 PairTune - 产品定位与功能设计（V1）

更新时间：2026-03-08

## 1. 品牌命名
- 中文主名：合拍
- 英文主名：PairTune
- 品牌语：两个人，更容易坚持
- 商店短描述：双人任务、积分激励、共同成长。

命名解释：
- 合拍：强调两人节奏一致，不限定情侣关系。
- PairTune：`Pair` 表示双人协作，`Tune` 表示调频和共振，语义与“合拍”一致。

## 2. 应用定位
- 类型：双人协作成长工具
- 面向关系：对象、闺蜜、室友、学习搭子、同事搭档
- 核心价值：把口头约定变成可执行、可激励、可复盘的双人协作闭环

## 3. 功能边界（V1）
- 任务：四象限任务、截止日期、积分奖励、完成状态
- 商城：发布奖励、兑换奖励、库存管理、已兑记录
- 通知：任务提醒、兑换提醒、关系动态、系统消息
- 我的：资料编辑、关系管理、账号安全、通知设置、反馈与隐私

## 4. 页面信息架构
- 底部导航：任务 / 商城 / 通知 / 我的
- 任务页：任务总览、筛选排序、四象限快捷创建
- 商城页：积分卡、可兑换商品、我发布商品、已兑记录
- 通知页：按消息类型聚合通知
- 我的页：账号与关系配置中心

## 5. UI 设计方向
- 视觉关键词：温暖、协作、成长、轻仪式感
- 设计原则：
  - 使用自定义品牌主题，不直接套用默认 Material 风格
  - 用分层渐变背景 + 卡片体系建立信息主次
  - 骨架屏采用 shimmer 动效，保持加载阶段连贯反馈
  - 保持高可读和高触控容错（移动端优先）
